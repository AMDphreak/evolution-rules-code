= Evolution Rules: Code
:revdate: 2026-03-22
:toc: left

Standard SDL rulesets for code migrations (e.g., PyQt5 to PyQt6, API shifts, etc.). This repository is the official code-level rules hub for the link:https://github.com/AMDphreak/evolution-engine[Evolution Engine] ecosystem.

This repository is a **Content Repository**. It contains the logic (rules) but requires the link:https://github.com/AMDphreak/evolution-engine[Evolution Engine] binary or the link:https://github.com/AMDphreak/evolution-engine-action[Evolution Engine Action] to execute them.

== What is here

* `rules/<language>/<library>/` — Versioned SDL rulesets organized by language and library/binding.

== Use with Evolution Engine (recommended)

The engine can use these rules from a local clone or pull them from Git:

[source,bash]
----
evolution-engine --path ./src --rules-repo https://github.com/AMDphreak/evolution-rules-code --library python/PyQt --from 5.15 --to 6.0
----

With a local checkout, point at the rules directory:

[source,bash]
----
evolution-engine --path ./src --rules-dir ./rules/python/PyQt --from 5.15 --to 6.0
----

See the link:https://github.com/AMDphreak/evolution-engine[Evolution Engine] repository for full options (remote rules, shortest path between versions, dry run, and more).
